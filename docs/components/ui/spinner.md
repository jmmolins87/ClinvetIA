# Spinner

Conjunto de indicadores de carga: `Spinner`, `LoadingOverlay`, `LoadingDots` y `LoadingRing`.

## Uso básico

```tsx
import { Spinner } from "@/components/ui/spinner"

<Spinner size="sm" variant="primary" />
```

## Cuándo usarlo

- Cargas de búsqueda
- Paginación
- Overlay temporal durante operaciones
