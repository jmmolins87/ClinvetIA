# Popover

Panel flotante no bloqueante para ayuda contextual o acciones breves.

## Uso básico

```tsx
import { Popover, PopoverContent, PopoverTrigger } from "@/components/ui/popover"

<Popover>
  <PopoverTrigger>Abrir ayuda</PopoverTrigger>
  <PopoverContent>Contenido</PopoverContent>
</Popover>
```

## Cuándo usarlo

- Ayuda contextual
- Paneles cortos
- Filtros compactos
