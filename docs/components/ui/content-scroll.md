# ContentScroll

Contenedor scrollable para listas largas dentro de paneles fijos.

## Uso básico

```tsx
import { ContentScroll } from "@/components/ui/content-scroll"

<ContentScroll className="max-h-96">
  {children}
</ContentScroll>
```

## Cuándo usarlo

- Tablas largas
- Logs
- Auditoría y paneles administrativos
