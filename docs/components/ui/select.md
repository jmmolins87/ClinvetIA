# Select

Select nativo estilizado con apariencia coherente con el design system.

## Uso básico

```tsx
import { Select } from "@/components/ui/select"

<Select defaultValue="demo">
  <option value="demo">Demo</option>
  <option value="admin">Admin</option>
</Select>
```

## Cuándo usarlo

- Formularios simples
- Filtros rápidos
- Estados o roles
