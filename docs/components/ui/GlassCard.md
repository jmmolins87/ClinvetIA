# GlassCard

Componente contenedor con acabado Liquid Glass para bloques principales del panel.

## Uso básico

```tsx
import { GlassCard } from "@/components/ui/GlassCard"

<GlassCard>
  <h3>Resumen</h3>
</GlassCard>
```

## Cuándo usarlo

- Tarjetas de dashboard
- Contenedores de formularios
- Secciones destacadas con borde y glow
