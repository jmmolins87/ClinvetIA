# Toggle

Botón con estado activo/inactivo para filtros o selección rápida.

## Uso básico

```tsx
import { Toggle } from "@/components/ui/toggle"

<Toggle pressed={active} onPressedChange={setActive}>
  Confirmadas
</Toggle>
```

## Cuándo usarlo

- Filtros por estado
- Vistas segmentadas
- Modos de selección directa
