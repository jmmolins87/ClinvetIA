# Toast

Sistema de notificaciones tipo toast con variantes y posiciones.

## Uso básico

```tsx
import { Toast, ToastDescription, ToastProvider, ToastTitle, ToastViewport } from "@/components/ui/toast"
```

## Uso recomendado

Para disparar notificaciones en aplicación, normalmente se usa `useToast` o `sonner`, dejando este componente como base visual.
