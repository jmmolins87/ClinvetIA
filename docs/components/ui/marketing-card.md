# MarketingCard

Tarjeta visual para bloques comerciales, beneficios o contenido de marketing.

## Uso básico

```tsx
import { MarketingCard } from "@/components/ui/marketing-card"

<MarketingCard title="Automatización" description="Reduce carga operativa" />
```

## Cuándo usarlo

- Landing pages
- Secciones de beneficios
- Casos de uso
