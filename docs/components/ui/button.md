# Button

Botón principal del sistema con variantes Liquid Glass y soporte `asChild`.

## Uso básico

```tsx
import { Button } from "@/components/ui/button"

<Button variant="default">Guardar</Button>
```

## Variantes comunes

- `default`
- `secondary`
- `accent`
- `warning`
- `destructive`
- `ghost`
- `outline`
- `link`
