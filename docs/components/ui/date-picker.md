# DatePicker

Selector de fecha o rango encapsulado para filtros y formularios.

## Uso básico

```tsx
import { DatePicker } from "@/components/ui/date-picker"

<DatePicker value={range} onChange={setRange} placeholder="Selecciona fecha" />
```

## Cuándo usarlo

- Filtros por rango
- Formularios de reserva
- Fechas administrativas
