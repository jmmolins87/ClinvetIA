# Card

Tarjeta base estructural con `CardHeader`, `CardContent` y `CardFooter`.

## Uso básico

```tsx
import { Card, CardContent, CardHeader, CardTitle } from "@/components/ui/card"

<Card>
  <CardHeader>
    <CardTitle>Título</CardTitle>
  </CardHeader>
  <CardContent>Contenido</CardContent>
</Card>
```

## Cuándo usarlo

- Contenido neutro
- Composición interna en tabs, dialogs o layouts
