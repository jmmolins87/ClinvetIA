# Table

Conjunto de subcomponentes para tablas del panel.

## Uso básico

```tsx
import { Table, TableBody, TableCell, TableHead, TableHeader, TableRow } from "@/components/ui/table"

<Table>
  <TableHeader>
    <TableRow>
      <TableHead>Nombre</TableHead>
    </TableRow>
  </TableHeader>
  <TableBody>
    <TableRow>
      <TableCell>Ana</TableCell>
    </TableRow>
  </TableBody>
</Table>
```

## Cuándo usarlo

- Datos administrativos
- Resúmenes operativos
- Tablas comparativas
