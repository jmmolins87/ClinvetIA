# Icon

Wrapper oficial para iconos Lucide con tamaños y variantes del sistema.

## Uso básico

```tsx
import { Zap } from "lucide-react"
import { Icon } from "@/components/ui/icon"

<Icon icon={Zap} variant="primary" size="lg" />
```

## Regla del proyecto

Todos los iconos de aplicación deben pasar por este componente.
